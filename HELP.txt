def, a bash-based personal encyclopedia system.

Usage:
def <word>:
    get the definition of a word.

def -af <word> <definition file>:
    add a definition item from a file.
    * the definition files are stored in ./data/def_files by convention.
    * the definition files are postfixed with .def by convention.

def -as <word>:<definition string>:
    add a definition item directly

def -h/def -help:
    display help doc

def -v/def -version:
    display the version
